Functions and Constants

The branch functions include common functions used throughout the library. Constants include specific constants used by MML.

Constants

Specific MML constants include eps which is a global constant set to 1e-3 used in the pressure loss equations and Q_eps which is set to 1e-3 that is used for continuous mass flow reversal.

Functions

Thermo square

The thermo square is used to compute the square of a flow rate in pressure loss computations.

f(x) = x*abs(x) if x> eps else x*eps

VaporMassFraction

For a 2 phase medium, this function computes the vapor mass fraction x of the medium, with the pressure and the enthalpy.

First it computes the vapor enthalphy hv (more precisely the dew enthalpy, which means a medium that is mostly vapor) and the liquid enthalpy hl (more precisely the bubble enthalpy, which is the enthalpy of the medium when it's mostly water but with a few vapor bubbles).

If h is over hv then the medium is complete vapor, so x=1. If it is under hl, then x=0.

If the enthalpy is between hl and hv then the medium is partially vaporized and the mass fraction is :

x=(h−hl)/(hv−hl)x = (h-h_l)/(h_v-h_l)x=(h−hl​)/(hv​−hl​)﻿

Power Heat Exchange

TSP handbook : Chapter 9.2.6

See also : NTU Method on Wikipedia

This function aims at computing the power exchanged in a heat exchanger using the Number of Transfer Units (NTU) method. This methos uses 2 steps : it first computes a maximum possible exchange Wmax, and then a coefficient epsilon called effectiveness so that the real power exchange is epsilon*Wmax.

Wmax is found by first identifying the minum heat capacity QCPmin, either Q_cold*Cp_cold or Q_hot*Cp_hot. The maximal exchange is then

Wmax=(QCp)min∗(Th,in−Tc,in)W_{max} = (QC_p)_{min} * (T_{h,in} - T_{c,in})Wmax​=(QCp​)min​∗(Th,in​−Tc,in​)﻿

The effectiveness is then computed as a function of the heat capacity ratio Cr = CPmin/CPmax and the number of transfer units NTU.

Cr=Cp,min/Cp,maxC_r = C_{p,min} / C_{p,max}Cr​=Cp,min​/Cp,max​﻿

NTU=Kth∗S/Cp,minNTU = K_{th} * S / C_{p,min}NTU=Kth​∗S/Cp,min​﻿

where Kth is the global exchange coefficient, and S the exchange surface.

The situation differs according to the thermodynamic conditions in the hot side. In all cases, the cold side is assumed to be monophasic. A variable HotPhase is used to discriminate two cases : either the hot side is monophasic or biphasic.

If the hot side is biphasic, then the minimum heat capacity is that of the cold side, Q_cold*Cp_cold. In this case the effectiveness does not depend on the flow configuration and is equal to :

ϵ=1−exp⁡(−NTU)\epsilon = 1 - \exp(-NTU)ϵ=1−exp(−NTU)﻿

If the maximum heat capacity is associated to the hot side (mixed fluid), then
